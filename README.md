Aproveitando que estou com mais tempo resolvi concluir um projeto que tinha iniciado e que precisava concluir, por fim hoje finalizei um dashboard interativo para monitoramento de margem negativa, projetado para identificar com precisão onde estão os gargalos de rentabilidade. 

A ideia é simples... permitir que o gestor pare de minerar dados e comece a tomar decisões em tempo hábil.

​

O Dashboard contém:



-​KPIs Estratégicos: Visão clara de Impacto Total, Venda Negativa e Recomposição de Margem.

​-Navegação Inteligente (Drill-down): Ao selecionar uma loja ou comprador, todo o dashboard se reajusta automaticamente, filtrando os dados em tempo real.

​-Detalhamento de Itens: Uma tabela dinâmica que revela exatamente quais produtos estão gerando o prejuízo, facilitando a negociação e o ajuste de estratégias.

​

Para construir essa solução, utilizei Python e o pandas, enquanto a interface e a reatividade dos gráficos ficaram por conta do framework Dash e da biblioteca Plotly. O resultado é um dashboard que processa grandes volumes de informação e os entrega de forma visual e intuitiva.
